feat: ✨ add `sd-skeleton` CSS style component

- Introduces `sd-skeleton` CSS style. Apply the class directly to any content element to display a loading placeholder that automatically matches the element's natural dimensions.
- Supports `sd-skeleton--animated` modifier for opt-in pulse animation.
- Supports `--sd-skeleton-color` CSS custom property for color customization.
- Deprecates the `sd-skeleton` web component in favor of the new CSS style.
