Introduction of the new Solid Design System MCP package, which includes:
- tool entry points for components, styles, templates, quickstart, token info, CD toolbox, icon search, and version metadata;
- package-level documentation support and metadata-driven build outputs for MCP tools;
- improved metadata quality for documentation consumers by updating component descriptions and adding summaries for all styles.
