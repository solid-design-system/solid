## Overview

`sd-mark` style utility

## Guidelines

### Use Cases

- To highlight text sections in a headline or displays.

### Rules

### Styling

- The font weight of the mark is defined by the headline/display styling used.
- Only use the styling for short text sections or combinations of words.

### Background Color

- Use light background options like white, neutral-100, or primary-100, or use a primary background when inverted.

### Accessibility

- For the green accent color in the mark, ensure a minimum font size of 18.67 pixels when bolded and 24 pixels when in regular weight.

### Related Templates

- mark--docs

Use the templates tool (with `template` arg) to retrieve the full code for any of these templates.
