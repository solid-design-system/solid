## Overview

`sd-copyright` style utility

## API

### Examples

Use the styles tool (with `style` + `example` args) to retrieve the HTML for any of these examples:

- sd-copyright/variants
- sd-copyright/no-shadow
- sd-copyright/orientation
- sd-copyright/placement

## Guidelines

### Use Cases

- Indicate the legal rights or usage terms for creative assets or articles.

### Rules

### Content and Readability

- Keep the text concise (ideally one line) and limit it to the name of the rights owner.
- Use the “&--shadow” property to ensure readability over images, videos or other visually complex backgrounds.

### Accessibility

- Choose the white variant with shadow for image backgrounds, and black or white for simpler or high-contrast backgrounds, or images with gradient overlays to ensure readability.
- Use the top placement when displayed in a video.

### Related Templates

- media--docs
- video--docs
- teaser-media--docs

Use the templates tool (with `template` arg) to retrieve the full code for any of these templates.
