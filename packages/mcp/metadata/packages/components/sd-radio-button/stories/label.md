Use the `default` slot to add a label.

```html
<div class="flex gap-12">
  <sd-radio-group value="1">
    <sd-radio-button value="1">
      <div>Label</div>
    </sd-radio-button>

    <sd-radio-button value="2">
      <div>Label</div>
    </sd-radio-button>

    <sd-radio-button value="3">
      <div>Label</div>
    </sd-radio-button>
  </sd-radio-group>
</div>
```
