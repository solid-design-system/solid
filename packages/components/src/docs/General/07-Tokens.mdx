## Intro

The Solid Design System includes two distinct token sets:

- **UI Core Tokens:** This is where all the fundamental design values are stored. They should never be used directly.
- **UI Semantic Tokens:** In this set, core tokens are referenced as aliases and come with specific usage contexts. It's essential to use tokens exclusively from the semantic token set.
