# Migration Guide: From `ui-tab-panel` to `sd-tab-panel`

The new `sd-tab-panel` is designed to replace the `ui-tab-panel`. Instead of mainly providing content via attributes, the `sd-tab-panel` component uses slots to allow for more flexibility and customization.

## ⚙️ Attributes

### ✨ New Attributes

#### [name]

The tab panel's name.

<hr />

## ✍️ CSS Variables

### ✨ New CSS Variables

#### [--padding]

The tab panel's padding.

<hr />

## 🧪 Methods

### ❌ Removed Methods:

The following methods have been removed from the new sd-tab-panel component:

1. [activate]
2. [deactivate]
