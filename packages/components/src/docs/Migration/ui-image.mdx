# Migration Guide: `ui-image`

The new `sd-media` and `sd-copyright` are designed to replace the `ui-image`. Rather than offering a complex component structure, they primarily focus on providing CSS styles.

Media no longer requires the `ui-image` wrapper and instead can directly utilize the provided CSS classes.

## üé® CSS Classes

### ‚ú® New CSS Classes

#### [sd-media]

The `sd-media` class is a utility meant to apply foundational styles to figure elements that act as media containers. It resets margins and provides text styling for any captions.

```html
<figure class="sd-media">
  <img ... />
  <figcaption>Your media caption</figcaption>
</figure>
```

#### [sd-copyright]

The `sd-copyright` is used to add a copyright notice overlay to media elements. It positions the element and defines padding, font size, color, and text shadow. It uses a CSS custom property `--copyright` for the text content.

```html
<div class="sd-copyright" style="--copyright: Your copyright text';">
  <img ... />
</div>
```
