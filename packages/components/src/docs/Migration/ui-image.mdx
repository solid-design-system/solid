# Migration Guide: `ui-image`

The ui-image has been transformed to `sd-media` and `sd-copyright`.

# ui-image to sd-media

Instead of mainly providing content via attributes, the new `sd-media` component uses slots to allow for more flexibility and customization.

## 💾 Slots

### ✨ New Slots

#### [default]

Content of the element.

<hr />

## ⚙️ Attributes

### ❌ Removed Attributes

The following attributes have been removed from the new sd-media component:

1. [altText]
2. [componentWidth]
3. [copyrightPosition]
4. [copyrightText]
5. [lazyLoading]
6. [lightbox]
7. [lightboxBackground]
8. [maskPosition]
9. [masked]
10. [mouseoverText]
11. [portrait]
12. [positionX]
13. [positionY]
14. [ratio]
15. [replace]
16. [sizes]
17. [src]
18. [subText]
19. [subTextHeadline]

<hr />

## 🥳 Events

### ❌ Removed Events:

The following events have been removed from the new sd-media component:

1. [componentResized]

<hr />

## 🧪 Methods

### ❌ Removed Methods:

The following methods have been removed from the new sd-media component:

1. [onAfterResize]

<hr />
