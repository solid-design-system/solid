# Migration Guide: From `ui-meta-info` to `sd-meta`

The new `sd-meta` is designed to replace the `ui-meta-info`. Rather than offering a complex component structure, they primarily focus on providing CSS styles for meta information. Make sure to provide correct semantics yourself, e. g. `<date>`.

## 🎨 CSS Classes

### ✨ New CSS Classes

#### [sd-meta--size-sm]

This class sets a smaller size for the meta element. You can use this option to accomodate smaller container/screen sizes.

#### [sd-meta--inverted]

This class inverts the meta element.

#### [sd-meta--pipe]

This class adds a pipe to the right of the meta element.

#### [sd-meta--light]

This class makes the sd-meta appear lighter.
