# Migration Guide: From `ui-map-marker` to `sd-map-marker`

The new `sd-map-marker` is designed to replace the `ui-map-marker`.

## 💾 Slots

So far no new slots were added to the `sd-map-marker` component.

<hr />

## ⚙️ Attributes

So far no new attributes were added nor were attributes removed from the `sd-map-marker` component.

## 🎨 CSS Classes

### 🔀 Updated CSS Classes

#### [--component-map-marker-scaling]

The css class `--component-map-marker-scaling` has been renamed to `--map-marker-scaling`.
