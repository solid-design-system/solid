# Migration Guide: `ui-container`

The ui-container has been transformed to `sd-container` and `sd-brandshape`.

# ui-container stylized="true" to sd-brandshape

The new `sd-brandshape` is designed to replace the stylized variant of `ui-container`. The `sd-brandshape` component uses slots to allow for more flexibility and customization.

## 💾 Slots

### ✨ New Slots

#### [default]

The content inside the brandshape.

<hr />

## ⚙️ Attributes

### ✨ New Attributes

#### [shapes]

Defines which shapes of the brandshape should be displayed from 'top', 'middle' and 'bottom'.

### 🔀 Updated Attributes

#### [variant]

Currently the color variants are reduced for the stylized container and it is in discussion whether more variants will be added in the future.

### ❌ Removed Attributes

The following attributes have been removed from the new sd-brandshape component:

1. [full-width]
2. [no-padding]
3. [opaque]
4. [stylized]

<hr />

3. [--component-container-outline-color-border]
4. [--component-container-outline-color-font]
5. [--component-container-primary-color]
6. [--component-container-primary-color-background]
7. [--component-container-primary-color-light]
8. [--component-container-primary-color-light-background]
9. [--component-container-secondary-color]
10. [--component-container-secondary-color-background]
11. [--component-container-white-color-background]
12. [--component-container-white-color-font]
