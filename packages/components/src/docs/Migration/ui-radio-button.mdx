# Migration Guide: From `ui-radio-button` to `sd-radio-button`

The new `sd-radio-button` is designed to replace the `ui-radio-button`. Instead of mainly providing content via attributes, the `sd-radio-button` component uses slots to allow for more flexibility and customization.

### [sd-radio-button-group] is now part of [sd-radio-group]

`sd-radio-group` is used to group multiple radios or radio buttons so they function as a single form control. Radio groups include a set of radios in which only one radio can be selected at a time. A group label can be displayed additionally.
