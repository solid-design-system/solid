# Migration Guide: From `[ui-audio-player]` to `[sd-audio]`

The new `[sd-audio]` is designed to replace the `[ui-audio-player]`. Instead of mainly providing content via attributes, the `[sd-audio]` component uses slots to allow for more flexibility and customization.
