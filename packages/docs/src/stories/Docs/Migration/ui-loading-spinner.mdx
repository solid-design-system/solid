# Migration Guide: From `ui-loading-spinner` to `sd-spinner`

The new `sd-spinner` is designed to replace the `ui-loading-spinner`. Instead of mainly providing content via attributes, the `sd-spinner` component uses slots to allow for more flexibility and customization.
