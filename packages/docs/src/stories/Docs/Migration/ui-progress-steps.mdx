# Migration Guide: From `[ui-progress-steps]` to `[sd-step-group]`

The new `[sd-step-group]` is designed to replace the `[ui-progress-steps]`. Instead of mainly providing content via attributes, the `[sd-step-group]` component uses slots to allow for more flexibility and customization.
