# Migration from v5

The v5 release introduces support for multi-theming tokens.

> This is work in progress.

## SCSS Token Naming

We improved our SCSS tokens to match them with our CSS variables:

**Before:**

```scss
.my-special-element {
  background-color: $sd-color-backgroundPrimary-100;
  color: $sd-color-textPrimary-DEFAULT;
}
```

**After:**

```scss
.my-special-element {
  background-color: $sd-color-background-primary-100;
  color: $sd-color-text-primary;
}
```
