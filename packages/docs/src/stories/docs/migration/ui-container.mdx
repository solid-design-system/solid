# Migration Guide: `ui-container`

The `ui-container` has been transformed and split up into to `sd-container` and `sd-brandshape`. Here you can find migration guides for both.

# ui-container stylized="true" to sd-brandshape

The new `sd-brandshape` is designed to replace the stylized variant of `ui-container` and uses slots to allow for more flexibility and customization.

## âš™ï¸ Attributes

### âœ¨ New Attributes

#### [sd-container--triangle-...]

Defines an optional triangle cut-out for sd-container with the following options: `default`, `top`, `right`, `bottom` or `left`. This allows for an indentation resembling an arrow on any side of the container.

### ğŸ”€ Updated Attributes

#### [noPadding]

The `noPadding` attribute has been replaced with the `sd-container--padding-...` attribute. The new `sd-container--padding-...` attribute can have the following options: `default` or `sm`.

#### [variant]

The `variant` attribute has been replaced with the `sd-container--variant-...` attribute. The `sd-container--variant-...` attribute can have the following options: `default`, `bg-primary-100`, `primary`, `border-neutral-400` or `white`.
