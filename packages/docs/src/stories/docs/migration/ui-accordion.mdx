# Migration Guide: From `ui-accordion` to `sd-accordion-group`

The new `sd-accordion-group` is designed to replace the `ui-accordion`.

## Attributes

### Updated Attributes

#### [auto-close]

The `auto-close` attribute has been renamed to `close-others` to better describe its functionality.
